# Stock Forum Complete Scaffold (Monorepo)

This scaffold contains backend (Node/Express/Sequelize+SQLite), frontend (React + Vite), and an optional Python worker.
It's designed as a full-feature starting point: authentication, posts, comments, uploads, sockets, and admin endpoints are scaffolded.

Quick start (backend):
1. cd backend
2. npm install
3. cp .env.example .env and set values
4. npm run migrate
5. npm run dev

Quick start (frontend):
1. cd frontend
2. npm install
3. npm run dev
